# Portfolio Fotografi Nahel Ahya Ramadhani

Website portfolio fotografi pribadi yang menampilkan koleksi foto dalam kategori
Portrait, Travel, Street, dan lainnya. Dibangun dengan HTML, CSS, dan JavaScript
untuk pengalaman yang responsif dan interaktif.

## Fitur

- **Hero Section**: Perkenalan singkat dengan foto profil.
- **Tentang**: Deskripsi diri dan keahlian fotografi.
- **Koleksi Foto**: Carousel interaktif menggunakan Splide.js dengan 8 foto.
- **Lightbox Preview**: Klik foto untuk melihat dalam ukuran penuh dengan
  navigasi gallery.
- **Footer**: Tautan sosial dan kontak langsung ke WhatsApp.
- **Responsif**: Desain yang menyesuaikan dengan berbagai ukuran layar.
- **Cursor Glow**: Efek kursor yang mengikuti mouse.

## Teknologi

- **HTML5**: Struktur halaman.
- **CSS3**: Styling dengan grid, flexbox, dan animasi.
- **JavaScript**: Interaktivitas navbar, Splide carousel, dan Lightbox.
- **Splide.js**: Library carousel untuk navigasi foto.
- **Lightbox2**: Library untuk preview gambar.
- **Font Awesome**: Ikon sosial.
- **Google Fonts**: Font Oswald.
